「404」404 Not Found。


  1. 「404」はページが見つからない事を表すHTTPステータスコードです。
  2. 入力欄に入力した単語のページに移動するはずでした。
  3. GitHub Pageでは404エラーが発生するとこの「404.html」を表示します。
  4. 単語が思いつかない、その他不具合などの疑いがある場合、
    入力欄に「とりせつ」「よんもじ」と入力してみて動作確認してみてください。

語彙が増えたら404に案内所設置予定。

サンプルページ。

とりせつ
「約四文字」の「取扱説明書」略して「取説(とりせつ)」。
よんもじ
